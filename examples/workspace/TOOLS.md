# Available Tools

This agent has access to various tools and capabilities. Use them wisely to accomplish tasks.

## Code Operations

- **File Management**: Read, write, edit, and organize code files
- **Search**: Find files and code patterns across the codebase
- **Version Control**: Git operations (status, diff, commit, push, PR creation)
- **Package Management**: Install dependencies, run build scripts

## Execution

- **Shell Commands**: Execute terminal commands for builds, tests, and operations
- **Process Management**: Run and monitor background processes
- **Testing**: Execute test suites and interpret results

## External Communication

- **Web Search**: Research best practices, documentation, and solutions
- **Web Fetch**: Retrieve and analyze web content
- **API Calls**: Integrate with external services

## Discord Integration

When operating as a Discord bot, you can:

- Respond to mentions and direct messages
- React to messages with emojis
- Send formatted messages with embeds
- Track conversation history per user/channel

## Best Practices

1. **Read before writing**: Always examine existing code before making changes
2. **Test changes**: Validate modifications with tests or builds
3. **Commit thoughtfully**: Create meaningful commits with descriptive messages
4. **Document decisions**: Explain non-obvious choices in code comments
5. **Handle errors gracefully**: Consider edge cases and error scenarios

## Limitations

- Cannot access private external APIs without credentials
- Cannot modify files outside the workspace
- Cannot execute destructive operations without confirmation
- Cannot access the user's personal data or system files
